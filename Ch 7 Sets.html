üéØ Elements of a Set

Great job getting here! Now let's learn about what we put INSIDE our sets. These are called ELEMENTS!
The objects that belong to a set are called its ELEMENTS.
üìù Sets are written with CAPITAL letters: A, B, X, M
üìù Elements are written with small letters: a, b, h, l, n

How to Write a Set:

We write sets using curly brackets { } with commas between elements:

A = {a, e, i, o, u}

We can write the elements in ANY order! These are all the SAME set:

A = {a, e, i, o, u}
A = {i, o, u, a, e}
A = {u, o, i, a, e}

üìù Three Ways to Write Sets

Awesome progress! Now I'll show you THREE different ways to write the same set. It's like having three different ways to describe your favorite ice cream! üç¶

Method 1: Roster or Tabular Method

We LIST all the elements separated by commas

Example: The set of odd natural numbers less than 15

O = {1, 3, 5, 7, 9, 11, 13}

Method 2: Set Builder or Rule Method

We write a RULE that describes the elements

Example: The set of odd natural numbers less than 15

O = {x : x is an odd natural number less than 15}

The colon : means "such that"

We read this as: "O is the set of all x such that x is an odd natural number less than 15"

Method 3: Description Method

We DESCRIBE the set in words

Example: The set of odd natural numbers less than 15

O = {Odd natural numbers less than 15}
All three methods describe the SAME set! Choose whichever is easiest for your problem.

üî¢ Cardinal Numbers

You're doing amazing! Now let's learn how to COUNT the elements in a set. This is called the CARDINAL NUMBER!
The Cardinal Number of a set is the number of DISTINCT (different) elements in it.
We write it as: n(A) = number of elements
We read it as: "n of A" or "cardinal number of set A"

Examples:

Set A = {5, 10, 15, 20, 25}

How many elements? Let's count: 5, 10, 15, 20, 25 = 5 elements

n(A) = 5

Tricky Example!

Set K = {x : x is a letter of the word 'HAPPINESS'}

First, let's write in roster form:

K = {H, A, P, I, N, E, S}

Notice: We only write each letter ONCE, even if it appears multiple times in the word!

Count: H, A, P, I, N, E, S = 7 letters

n(K) = 7
